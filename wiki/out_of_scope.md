# Out of Scope — Phase 2

Items explicitly excluded. Record reasons so future sessions don't relitigate them.

| Item | Reason |
|------|--------|
| O(2)-equivariant model | Discontinued — OOM issues on available hardware; only 2/4 data fractions completed in Phase 1 |
| SO(2) rerun at higher frequency | Deprioritized — discrete groups (D4, C8) outperformed SO(2) in Phase 1; marginal value vs cost |
| App / operational deployment | After job search; not a research contribution |
| Full Cascades dataset | Pending Dallas/NWAC collaboration — no data yet |
| Volume estimation (DEM differencing) | Would require pre/post DEM, not available for all scenes |
| CROCUS snowpack model | French model, less relevant for Norway; SeNorge is the appropriate choice |
| SAR backscatter-only D-scale prediction | No physical basis without geometry; area is the primary proxy |
| New SAR scenes beyond AvalCD | Out of budget/scope; single Tromsø scene for size experiment is the limit |
| Multi-temporal time series | Phase 3 territory |
| 3D avalanche dynamics modeling | Not a remote sensing problem |
